What changed, by area

Security high impact

  • Dependency advisories 18 → 1 (0 high/critical; the remaining is a dev-server-only Low). Cleared ws, react-router-dom, vite, dompurify, lodash, shell-quote (critical), and dropped xlsx (replaced by the lighter write-excel-file).

  • Guided-map SVG is now DOMPurify-sanitized (stored-XSS closed).

Sync medium

  • Legacy pull path now runs its config/L2 side-pulls (it was skipping them entirely).
  • Background sync now starts in-process (was a single un-retried loopback fetch that, if it failed, meant sync never started).
  • Config writes are no longer a fire-and-forget unhandled rejection; delta broadcasts coalesced into one message (was one per row → flooded the UI on large syncs).

  • Corrected stale docs/comments.
