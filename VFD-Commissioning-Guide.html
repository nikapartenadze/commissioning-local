VFD Commissioning Wizard Guide

Before You Start

  • PLC is connected — green indicator in the top toolbar
  • IOs have been pulled — use Pull IOs in PLC Config if not done
  • FV data is loaded — FV tab shows VFD sheets with devices listed
  • VFD is powered on — the physical drive must have power
  • Correct device identified — match the on-screen name to the physical label
Safety The wizard commands the VFD to jog and run the motor. Ensure the area around the belt and motor is clear of personnel and tools before Steps 3 and 4.

The FV Page

Click FV in the top navigation. The Functional Validation page shows all device sheets, search, and filter controls.

FV page showing the VFD sheet (1022 devices). Sheet tabs along top, search and filter bar, device grid with HP, identity, tracking, and speed columns.

Opening the VFD Wizard

  1. On the FV page, select the VFD sheet tab.
  2. Use the search bar to find your device (e.g., type "NCP1").
  3. Hover over the device name in the first column — a tooltip appears.
  4. Click the device name to open the wizard.
Hovering over a device name shows "Open VFD wizard for NCP1_1_VFD". Click to open.
Note Only devices on sheets named "VFD" or "APF" show the wizard button.

Step 0 — VFD Online

0

VFD Online Check

Confirms the VFD is powered on and ready to accept commands.

Automatic — no action required. The wizard checks the PLC and auto-advances to Step 1 when the VFD is detected online.

Step 0: VFD Online check. Left sidebar shows all 6 steps (1–5 locked). Bottom shows PLC status and navigation.
Stuck on "Waiting"? Verify VFD has power, the PLC program has the CBT AOI for this device, and check the drive display for fault codes.

Step 1 — Identity Check

1

Identity Check

Proves you are at the correct physical VFD.

  1. Walk to the physical VFD shown on screen.
  2. Press F1 on the VFD's three-button keypad.
  3. The wizard detects the press — F1 indicator turns green.
  4. Confirmation prompt: "Did you just press F1 on [DEVICE]?"
  5. Tap "Yes, I pressed it".
Recorded "Verify Identity" column → your initials + date (e.g., ASH 4/21)

Step 2 — Horsepower Check

2

Horsepower Check

Confirms motor and drive HP ratings match specification.

  1. Read Motor HP from the motor faceplate (physical label).
  2. Read Drive HP from the VFD label (on enclosure).
  3. Enter both values in the input fields.
  4. Click "Confirm HP".
Recorded "Motor HP (Field)" and "VFD HP (Field)" columns are filled with the values you entered.
Mismatch? If Motor HP and Drive HP don't match, stop. Report to the commissioning lead.

Step 3 — Bump Test

3

Bump Test (Direction Check)

Jogs the motor briefly to verify rotation direction.

Safety — Clear the area first The motor will physically run. Ensure belt area is clear. Notify nearby workers.
  1. Click "Bump Motor" — motor jogs ~1 second, then stops.
  2. Mechanical team observes belt/motor direction.
  3. If correct: click "Yes — Confirm Direction".
  4. If wrong: do NOT confirm. Fix phase wiring, then bump again.
Recorded "Ready For Tracking" column → your initials + date

Step 4 — Verify Controls

4

Verify Controls

Confirm VFD keypad controls (F0/F1/F2) are working before handing over to the mechanical team.

Verify these keypad controls:

F0
Speed Down
F1
Start / Stop
F2
Speed Up
  1. Test F1 — verify motor starts and stops.
  2. Test F2 / F0 — verify speed increases and decreases. Watch live RVS on screen.
  3. Click "Controls Verified — Ready for Tracking".
Informational only This step does NOT write to the spreadsheet. Notify the mechanical team that the conveyor is ready for belt tracking.
Belt Tracked The mechanical team fills the "Belt Tracked" column manually (initials + date) when tracking is complete — this may happen days later.

Step 5 — Speed Calibration

5

Calibrate Speed

Measures belt speed to create the FPM-to-RVS calibration point.

Locked This step is locked until the mechanical team marks the "Belt Tracked" column (initials + date).
  1. Click "Run at 30 RVS" — VFD is set to 30 RVS.
  2. Wait for steady speed. Watch the live RVS reading on screen.
  3. Use a tachometer to measure belt speed in FPM.
  4. Enter FPM value, click "Log Speed".
Recorded "Speed Set Up" column → your initials + date + FPM and RVS values (e.g., ASH 4/21 · 450 FPM @ 29.99 RVS)

This is the final step. Click "Done" to close. All data syncs to cloud automatically.

Troubleshooting

ProblemSolution
Stuck on "Waiting for VFD"Check VFD power. Verify PLC has CBT AOI for this device. Check drive for faults.
F1 not detectedHold F1 for 1–2 seconds. Confirm correct physical VFD.
PLC Offline in sidebarReconnect PLC from main toolbar, reopen wizard.
Bump Motor does nothingPLC must be connected. Step 2 must be complete.
Speed shows "—"Check PLC program exposes the speed tag correctly.
Red error bannerPLC write failed. Check connection, retry.
FV page emptyUse "Pull FV from Cloud" button or full Pull IOs from settings.
Not syncing to cloudCheck cloud URL and API password in Settings.
